Animcação animate em CSS

A animação em CSS é uma técnica que permite criar movimentos e transições suaves em elementos HTML. Ela é amplamente utilizada para melhorar a experiência do usuário, tornando as interfaces mais dinâmicas e atraentes. A animação em CSS pode ser aplicada a diversos elementos, como texto, imagens, botões e outros componentes da página. Existem várias propriedades e técnicas para criar animações em CSS, incluindo keyframes, transições e transformações. Com a animação em CSS, é possível criar efeitos visuais impressionantes, como desvanecimentos, rotações, escalas e muito mais, sem a necessidade de usar JavaScript ou outras linguagens de programação.

Exemplo: 

        .box{
            width: 100px;
            height: 100px;
            background-color: red;
            animation: move 2s infinite;
            
        }
        @keyframes move {
            0% { transform: translateX(0); }
            50% { transform: translateX(100px); }
            100% { transform: translateX(0); }
        }
