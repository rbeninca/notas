# Planejamento das Aulas
## 1. Aula 1
### Fundamentos da Web e Redes
#### Tópicos
- Apresentação do Plano de Ensino da unidade curricular.
- História da Web: Evolução da Web 1.0 (páginas estáticas) para a Web moderna (SPAs e aplicações dinâmicas).
- Arquitetura Cliente-Servidor: Como navegadores e servidores se comunicam. Protocolos de Rede utilizados O papel do TCP/IP e DNS na navegação web.
- Protocolo HTTP e HTTPS: Métodos de requisição (GET, POST) e códigos de status (200, 404, 500).
- Ferramentas do Desenvolvedor (DevTools): Inspecionando a aba "Network" no navegador.

---

## 2. Aula 2
### HTML5 – Estrutura e Semântica
#### Tópicos
- Introdução ao HTML5 e sua importância no desenvolvimento web.
- Estrutura básica de um documento HTML.
- Principais tags HTML.
- Tags semânticas: header, nav, main, section, article, footer.
- Boas práticas de marcação e acessibilidade.
- Organização de conteúdo para SEO básico.

---

## 3. Aula 3
### Introdução ao CSS
#### Tópicos
- Conceito de CSS e separação entre conteúdo e apresentação.
- Sintaxe do CSS.
- Seletores básicos e combinadores.
- Propriedades de texto e cores.
- Box Model: margin, border, padding e content.
- Aplicação de estilos externos, internos e inline.

---

## 4. Aula 4
### CSS – Layout e Responsividade
#### Tópicos
- Conceito de layout na web.
- Flexbox: container, eixos, alinhamento e distribuição.
- Grid Layout: linhas, colunas e áreas.
- Design responsivo.
- Media Queries (@media screen).
- Adaptação de layouts para diferentes tamanhos de tela.

---

## 5. Aula 5
### Prática em Sala – HTML e CSS
#### Tópicos
- Revisão geral de HTML5 e CSS3.
- Estruturação de páginas completas.
- Aplicação de layout responsivo.
- Organização visual e hierarquia de informação.
- Correção de problemas comuns de estilização.
- Boas práticas de organização de arquivos.

---

## 6. Aula 6
### Avaliação 1
#### Tópicos
- Avaliação teórica sobre fundamentos da Web, HTML e CSS.
- Avaliação prática de construção e estilização de página web.
- Análise de estrutura, semântica e responsividade.
- Aplicação correta de seletores e propriedades CSS.

## 7. Aula 7
### Introdução ao JavaScript
#### Tópicos
- Introdução à linguagem JavaScript.
- Inclusão de scripts em páginas HTML.

- Variáveis (var, let, const).
- Tipos de dados.
- Estruturas condicionais.
- Funções.

## 8. Aula 8
### Manipulação do DOM e Eventos
#### Tópicos

- Interação dinâmica com o usuário.
