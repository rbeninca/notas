# Planejamento das aulas
## 1. Aula 1 
### Plano de Ensino
#### Tópicos
- plano de ensino
- Introdução ao a web
- Uso do git (init, add, status, commit, log) e github
- estrura básica HTML
### Objetivos
* Apresentar o plano de ensino e os conteúdos que serão abordados ao longo do curso.
* Sistema de Controle de Versão (GIT)
* Compreender a história das técnologias da web, motivação para desenvolvimento web,css,js.

## 2. Aula 2
### Web 
#### Tópicos
- História da web;
- Arquitetura Clinte Servidor    ( Cliente <-HTTP-> Servidor);
- Protocolo HTTP (Requisições e Respostas);
- Estrutura básica de uma página web;

## 3. Aula 3 
### HTML básico
#### Tópicos
- Tags HTML básicas (head, body, title, h1-h6, p, a img, div, span, etc.)
- Atributos comuns (id, class, href, src, alt, etc.)    
- Estrutura de um documento HTML (DOCTYPE, html, head, body)
- Semântica HTML (header, nav, main, section, article, footer)
- Formulários HTML (form, input, textarea, button, select, etc.)
- introdução a CSS (link, style, inline, etc.)

## 4. Aula 4
### Estilização com CSS
#### Tópicos
- Seletores e propriedades CSS
* Introduzir as ténclogias básicas da web (arquitetura cliente-servidor,html,http,js).
- Cores, fontes, border, background e unidades de medida
- Box model e layout
